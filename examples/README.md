# ComfyUI MCP Examples

This directory contains example projects demonstrating how to use the ComfyUI MCP server with Godot games.

## Examples

- **Character Portrait Generation**: Generate NPC portraits for RPGs
- **Item Icon Batch Generation**: Create unique item icons in bulk
- **Environment Texture Generation**: Generate tileable environment textures
- **Real-time Generation**: On-demand asset generation during gameplay
- **Procedural Sprite Variation**: Generate sprite variations procedurally

## Getting Started

1. Ensure ComfyUI is running locally on `http://127.0.0.1:8188`
2. Install the comfyui-mcp package: `pip install -e .`
3. Configure your MCP server in `.mcp.json`
4. Explore the examples in this directory

Each example includes:
- Godot project files
- ComfyUI workflow templates
- Integration code
- Documentation
